# Employee Leave Management System
A role-based MERN app for managing employee leaves, built with React, Express, MongoDB, and Material UI.

## ✨ Features
- Employee login/register
- Apply for leave
- View leave history
- Manager leave approvals
- Role-based access


## 🛠️ Tech Stack
- Frontend: React, React Router, Vite, Material UI
- Backend: Node.js, Express, MongoDB, Mongoose
- Auth: JWT
- State Management: Context API


## 📁 Folder Structure
- `/client` - Frontend
  - `components/`
  - `context/`
  - `pages/`
  - `services/`
- `/server` - Backend
  - `models/`
  - `routes/`
  - `controllers/`


## ⚙️ Getting Started

### Backend

cd backend
npm install
node server.js


### Frontend
cd frontend
cd frontend(2)
npm run dev


## 🔐 Environment Variables

In backend add .env file which contain three rows
PORT=5000 
MONGO_URI=your_mongo_connection in curly braces
JWT_SECRET=your_secret_key
